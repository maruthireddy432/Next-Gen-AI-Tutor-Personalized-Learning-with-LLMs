## Project Overview

The Adaptive AI Tutor System enhances learning by dynamically adjusting explanations, questions, and feedback based on student performance and interaction history.

Unlike traditional e-learning platforms, this system:

- Understands student queries contextually
- Adapts difficulty levels dynamically
- Provides personalized explanations
- Tracks learning progress
- Generates quizzes and assessments automatically
